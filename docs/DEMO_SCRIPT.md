# 7 分钟比赛演示脚本

## 固定案例

- 学生：`demo_basic`
- 输入：`我是计算机专业大二学生，Python 基础一般，函数、Pandas 和数据分析项目不熟，希望通过图解、代码案例和项目练习学习。`
- 主题：`Pandas 数据清洗与分析综合实践`
- 问题：`Pandas 里 dropna 和 fillna 应该怎么选择？`

## 时间安排

1. **0:00–0:35｜项目定位与真实性**
   - Dashboard 展示“面向高校 Python 课程的个性化 AI 学习智能体平台”。
   - 展示六阶段闭环。
   - 点击模型连接测试，说明 Provider、模型、Mock 与回退状态。
2. **0:35–1:15｜画像构建**
   - 切换 `demo_basic`。
   - 输入固定描述，生成八维画像。
   - 展示生成来源、SQLite 保存和画像变化。
3. **1:15–1:55｜课程知识库**
   - 展示 17 个课程主题和当前检索模式。
   - 检索固定问题，展示至少 3 条来源、片段、相关度。
4. **1:55–3:30｜多智能体资源生成**
   - 生成固定主题。
   - 展示七阶段卡片与当前 Agent。
   - 展示学习路径、讲义、思维导图、题库、代码、PPT 大纲、动态流程图。
   - 指出每项资源的 Agent、RAG、模型来源和人工核验状态。
   - 在“可执行学习计划”勾选一项，展示完成比例与剩余时间。
5. **3:30–4:10｜PPTX 与轻量多模态**
   - 下载 PPTX，打开标题或缩略图。
   - 展示 HTML 动态流程图。
   - 明确它是轻量多模态，不是真实视频。
6. **4:10–5:05｜RAG 智能辅导**
   - 提问固定问题。
   - 展示模型来源和检索模式。
   - 展开引用片段。
   - 说明低相关度时系统停止确定性回答。
7. **5:05–6:15｜学习评估**
   - 故意答错选择/判断，并提交简答和代码。
   - 展示精确判分、关键词 MVP、关键语句检查边界。
   - 展示错题知识点、建议与画像写回。
8. **6:15–7:00｜闭环结果**
   - 返回 Dashboard。
   - 展示持久化学习计划、最近测评、更新后的薄弱点、掌握度与学习轨迹。
   - 总结真实能力、MVP 边界和后续扩展。
