# 7 分钟比赛演示脚本

## 固定演示案例

- 课程：Python 程序设计与数据分析
- 学生输入：`我是计算机专业大二学生，Python 基础一般，函数、Pandas 和数据分析项目不熟，希望通过图解、代码案例和项目练习学习。`
- 学习主题：`Pandas 数据清洗与分析综合实践`

## 时间安排

1. **0:00–0:35｜真实性声明**
   - 打开 Dashboard。
   - 指出当前模型模式和检索模式。
   - 说明 Mock 只替代大模型文本，数据库、检索、流程、评分和画像回写真实执行。
2. **0:35–1:20｜画像生成**
   - 在学生画像页粘贴固定学生输入。
   - 展示八维抽取、SQLite 保存和“更新前/更新后”。
   - 快速切换三类演示学生，说明画像独立。
3. **1:20–1:55｜知识库与 RAG 检索**
   - 展示内置 Python 资料。
   - 搜索“Pandas 缺失值为什么不能总是删除”。
   - 展示 top-k 来源、片段和相关度，并说明当前为 MVP Hashing 检索。
4. **1:55–3:35｜多智能体资源生成**
   - 输入固定学习主题。
   - 展示七个流程状态：`profile_loaded` 到 `saved`。
   - 展示六类资源，并指出每类资源由哪个 Agent 生成。
   - 展示 ReviewAgent 的动态规则结果。
5. **3:35–4:30｜个性化差异**
   - 记录当前基础型学习路径。
   - 切换实践型或考试型学生，对同一主题重新生成。
   - 对比学习顺序、任务和内容风格的明显差异。
6. **4:30–5:20｜RAG 智能答疑**
   - 提问“Pandas 清洗缺失值应该怎么选择策略？”。
   - 展示检索引用、证据状态和回答。
   - 可提一个知识库外问题，展示“证据不足，需要人工核验”。
7. **5:20–6:25｜学习评估与画像写回**
   - 故意答错一道选择题和一道代码题。
   - 展示精确判分、关键词评分、代码关键语句检查及人工核验提示。
   - 展示薄弱点与错题写回画像的前后变化。
8. **6:25–7:00｜Dashboard 闭环**
   - 返回 Dashboard。
   - 展示最近测评、错题、薄弱点和当前学生资源历史已经变化。
   - 总结下一步为接入讯飞星火和 sentence-transformers。
