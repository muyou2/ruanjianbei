# AI Coding 工具使用说明

本项目开发过程中使用 OpenAI Codex 辅助完成代码骨架、重构建议、自动化测试、文档整理和本地运行排错。

## 使用范围

- 根据赛题需求生成前后端初始结构。
- 辅助实现 FastAPI、React、SQLite、Chroma 和测试代码。
- 协助检查功能宣传与真实实现是否一致。
- 辅助检索公开技术资料并整理来源。

## 人工责任边界

- 项目需求取舍、产品定位和最终提交内容由参赛团队确认。
- AI 生成代码需要经过本地测试、功能验收和安全检查后使用。
- API Key、学生隐私数据和未公开比赛材料不写入代码仓库。
- AI 输出不作为课程事实的最终依据；学术内容通过知识库引用、规则审校和人工核验控制风险。

## 可复现性

仓库保留依赖文件、环境变量示例、自动测试、一键启动脚本和功能状态表。无真实模型 API Key 时，系统会明确进入 Mock 演示模式，不把模板结果标记为真实模型输出。
