# 当前不足与后续扩展

## MVP 边界

- 简答题采用关键词覆盖度评分。
- 代码题仅检查关键语句，不执行学生代码。
- 掌握度是基于逐题得分的滚动计算，不是 DKT/BKT。
- 推荐策略使用画像、薄弱点、掌握度和反馈规则，尚非复杂推荐模型。
- 八智能体是可追踪协同流水线，尚无复杂 LangGraph checkpoint 与多轮自治协商。
- HTML 动态流程图属于轻量多模态资源，不是真实视频生成。

## 外部依赖风险

- 真实模型质量、价格和延迟取决于所配置 Provider。
- BGE 模型首次下载需要网络和磁盘空间；不可用时自动回退 Hashing。
- PDF 仅支持可提取文本，不包含 OCR。

## 待扩展能力

- 讯飞语音合成与真实视频生成。
- Docker 安全代码沙箱。
- 教师审核、发布和人工修订工作流。
- 登录、权限、隐私数据导出与删除。
- DKT/BKT 等知识追踪模型。
- 复杂 LangGraph checkpoint、恢复与人工中断。
- 生产级任务队列、监控和云部署。
