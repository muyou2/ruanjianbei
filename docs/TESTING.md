# 测试说明

## 自动测试

- 后端：画像抽取、文本切块、Hashing 向量、Review 风险提示。
- 前端：统一 API 响应解包。
- 构建：TypeScript 编译、Vite 生产构建。
- E2E：启动前后端后，在 `frontend` 执行 `npm.cmd run test:e2e`，复用系统 Microsoft Edge。

## 人工验收

1. 无 Key 启动，状态显示 Mock。
2. 输入自然语言并生成八维画像。
3. 确认内置三份 Python 资料已自动入库。
4. 生成六类资源，进度从 0 到 100。
5. 答疑展示至少一个引用片段。
6. 完成四道题，评估报告写回画像。
7. 上传错误格式时收到明确提示。
