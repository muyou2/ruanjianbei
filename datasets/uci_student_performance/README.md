# UCI Student Performance 数据集

- 官方页面：https://archive.ics.uci.edu/dataset/320/student+performance
- 作者：Paulo Cortez
- 许可证：Creative Commons Attribution 4.0 International（CC BY 4.0）
- 引入日期：2026-06-19

项目使用 `raw/student-por.csv` 中的 649 条记录和 30 个特征，展示学习行为分析、风险解释和数据适用边界。

## 使用边界

该数据来自葡萄牙两所中学的葡萄牙语课程，不是中国高校 Python 课程数据。本项目不会用它训练模型或自动决定学生权益，只将其作为公开、可复现的学习分析基准。

引用：

> P. Cortez and A. Silva. Using Data Mining to Predict Secondary School Student Performance. EUROSIS, 2008.
