# NumPy、Pandas 与数据分析项目

## 数据分析流程

一个可复现的数据分析项目通常包括：定义问题、获取数据、理解字段、清洗数据、探索分析、可视化、解释结论和记录限制。

结论必须与数据证据对应。相关关系不等于因果关系，样本偏差、缺失值和异常值都可能影响结论。

## NumPy 基础

NumPy 的核心对象是多维数组 `ndarray`。数组通常要求元素类型一致，支持向量化运算和广播。

```python
import numpy as np

scores = np.array([82, 91, 88, 76])
normalized = (scores - scores.mean()) / scores.std()
```

向量化代码通常比逐元素 Python 循环更简洁，但应先确认数组形状。广播错误常来自维度理解不清。

## Pandas 基础

`Series` 是带索引的一维数据，`DataFrame` 是二维表格数据。

```python
import pandas as pd

df = pd.DataFrame({
    "topic": ["基础", "函数", "Pandas"],
    "minutes": [45, 55, 70],
    "score": [82, 88, 91],
})
summary = df.groupby("topic", as_index=False)["score"].mean()
```

读取数据后应先检查 `head()`、`info()`、列名、类型、缺失值数量和重复行。

## 缺失值与异常值

缺失值处理没有统一答案。删除、填充或保留必须结合字段含义和缺失机制。均值填充可能缩小方差；类别字段可使用众数或新增“未知”类别。

异常值可能是录入错误，也可能是真实但罕见的现象。发现异常值后应回到数据来源验证，不应机械删除。

2. 统计各主题平均学习时长与成绩。
3. 找出多次尝试后仍低于 60 分的薄弱主题。

5. 说明分析限制，不能把相关性直接解释为因果。
